# First-Hackathon-Project
PROJECT TITLE- AI chatbot for college queries: An AI chatbot for college queries is a smart virtual assistant that helps students get instant answers about admissions, courses, placements, fees, and campus facilities. It provides 24/7 support, reduces manual workload, and improves communication between students and the institution.
Features -
a.24/7 instant response to student queries (admissions, courses, placements, fees, etc.)
b.Personalized answers based on user input
c.Easy navigation with quick reply options
d.Reduces workload on college staff
e.Can handle multiple users at the same time

Tech Stack-
a.Frontend: HTML, CSS, JavaScript
b.Backend: Python (Flask) / Node.js
c.Database: Firebase / MongoDB
d.AI/NLP: OpenAI API / Dialogflow
e.Hosting: Firebase Hosting / Heroku / Netlify

Setup Instructions-
a.Create project folder and set up frontend (HTML, CSS, JS files)
b.Build backend using Flask or Node.js
c.Connect database (Firebase or MongoDB)
d.Integrate AI chatbot API (OpenAI or Dialogflow)
e.Run the backend server locally
f.Test chatbot in browser
g.Deploy using Firebase / Heroku / Netlify
